# OGQ Desktop Pet

일할 때 옆에 두는 귀여운 데스크톱 펫

## 소개

OGQ Desktop Pet은 데스크톱에 귀여운 캐릭터를 띄워두는 Electron 기반 앱입니다. 키보드와 마우스 활동을 감지해서 캐릭터가 상태에 따라 다양한 반응을 보여주고, 업무와 휴식 시간을 자동으로 기록합니다.

## 주요 기능

### 캐릭터 시스템
- 6종 캐릭터 선택 (교모티콘, 하찮햄, 베리콘, 구북티콘, 혜냥이랑, 해달티콘)
- 캐릭터별 24개 GIF 애니메이션
- 상태에 따라 3초마다 다른 모션으로 자동 전환
- 드래그로 원하는 위치에 배치

### 자동 활동 감지
- 키보드/마우스 입력을 감지하여 자동으로 업무/휴식 판별
- 10분 미입력 시 자동으로 휴식 모드 전환
- 50분 연속 업무 시 "좀 쉬어!" 알림
- 10분 연속 휴식 시 "슬슬 할까?" 알림
- 상태 전환 시 캐릭터 말풍선 표시

### 활동 리포트
- 오늘 업무/휴식 시간 실시간 표시
- 주간 막대 그래프로 패턴 확인
- 일별 통계 자동 저장

### 타이머
- 집중/휴식 시간 직접 설정
- 시작/멈춤 제어
- 집중 끝나면 자동 휴식, 휴식 끝나면 자동 집중 반복

### 메모하기
- 여러 개의 메모 생성/삭제
- 사이드바에서 메모 목록 관리
- 자유 텍스트 입력, 자동 저장
- 앱 재시작해도 유지

### 기타
- macOS 트레이 아이콘 지원
- 항상 위 / 투명도 조절
- 마우스 커서 위치에서 캐릭터 시작
- Cmd+Shift+P로 펫 보이기/숨기기
- 시작 시 자동 실행 설정

## 설치

### DMG 설치 (macOS)
[Releases](https://github.com/khk15-prog/ogq-desktop-pet/releases) 페이지에서 최신 DMG 파일을 다운로드하여 설치합니다.

### 직접 빌드
```bash
git clone https://github.com/khk15-prog/ogq-desktop-pet.git
cd ogq-desktop-pet
npm install
npm start        # 개발 모드 실행
npm run build    # macOS 빌드 (universal)
```

## 사용법

1. 앱을 실행하면 캐릭터 선택 창이 열립니다
2. 원하는 캐릭터를 클릭하면 데스크톱에 펫이 나타납니다
3. 캐릭터를 우클릭하면 메뉴가 열립니다
   - 메모하기 / 활동 리포트 / 타이머 / 캐릭터 변경 / 종료
4. 우측 상단 상태 카드를 클릭하면 활동 리포트를 볼 수 있습니다
5. 캐릭터를 드래그해서 원하는 위치로 이동할 수 있습니다
